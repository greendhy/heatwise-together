# HeatWise Together

A free online climate-risk card generator for older adults, caregivers, and community volunteers.

The prototype runs entirely in the browser. It can pull current public weather data from Open-Meteo by city search or browser location, then generate a large-type action card for heat, smoke, heavy rain, or heat-plus-power-outage risks.

Built as a small public climate communication pilot.

Live demo:

https://greendhy.github.io/heatwise-together/
